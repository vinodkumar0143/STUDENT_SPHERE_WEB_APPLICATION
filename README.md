🎓 Student Sphere

A Full-Stack Student Collaboration & Learning Platform

Student Sphere is a centralized platform designed to help students manage learning resources, share notes, discover opportunities, collaborate with peers, and build professional resumes from a single dashboard.

🚀 Features
🔐 Authentication System
 .User Registration
 .User Login
 .JWT Authentication
 .Secure Password Hashing using bcryptjs
 .Forgot Password Functionality
 
📚 Notes Hub

 .Academic Resource Management
